## CRUD Functionality

- **Create** – Users can add new medicines with name, quantity and expiration date.
- **Read** – Users can view a list of all medicines in the inventory.
- **Update** – Users can edit medicine details such as quantity or expiration date.
- **Delete** – Users can remove medicines from the inventory.

## Authentication

- User registration and login
- Secure access to user-specific data

## Tech Stack
- HTML, CSS, JavaScript
- Supabase (Authentication & Database)
